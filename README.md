# 📊 E-commerce Sales Performance & Revenue Analysis
*Transforming raw transaction data into actionable business intelligence using Python.*

## 📖 Project Overview
This project performs an end-to-end **Exploratory Data Analysis (EDA)** on an e-commerce dataset. The goal is to identify sales trends, analyze customer preferences in sizing and color, and detect high-revenue outliers to optimize business operations.

## 🛠️ Tech Stack & Libraries
- **Language:** Python
- **Data Manipulation:** Pandas, NumPy
- **Data Visualization:** Matplotlib, Seaborn
- **Environment:** Jupyter Notebook / Google Colab

## 🚀 Key Features
- **Data Cleaning:** Handled missing values and verified dataset integrity (0 duplicates).
- **Time-Series Analysis:** Tracked monthly revenue growth from June to September.
- **Outlier Detection:** Used the **Interquartile Range (IQR)** method to isolate high-value orders.
- **Visual Storytelling:** Developed charts for monthly revenue and product size distribution.

## 📈 Major Insights
- **Growth:** Monthly revenue increased by **~84%** over four months.
- **Demand:** **XL** is the most popular size, accounting for **33.5%** of all sales.
- **Top Product:** Identified SKU `799` as the best-selling item by quantity.
